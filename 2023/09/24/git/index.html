Git

Git是一个分布式版本控制工具，主要用于管理开发过程中的源代码文件（java类，xml文件，html页面等）

Git常用指令

git clone:克隆 git init:本地初始化一个仓库

git add：将文件增加到缓存区 git commit:将文件提交到版本库当中

git reset:将缓存区文件取消缓存或者切换到指定版本 git log:查看日志
